# qSCMA
Quantile Single-cell Metallomic Analysis - a R package to:

link health effect to single-cell metallomic features, including quantile, heterogeneity index, mean value, and prevalence index.

Developed by Guohuan Zhang

Dowload the qSCMA package with following lines:

  if(!require(devtools)){
    install.packages("devtools")
  }
  
  devtools::install_github('Thuoyuing/qSCMA')
